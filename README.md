# TaskFlow Pro - Advanced Task Management System

A modern, full-stack task management application built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- 🔐 **Authentication**: Secure JWT-based authentication
- 📊 **Kanban Board**: Drag-and-drop task management
- 📈 **Analytics**: Visual insights with charts and graphs
- 🎨 **Dark/Light Mode**: Seamless theme switching
- 📱 **Responsive Design**: Works on all devices
- ⚡ **Real-time Updates**: Instant task synchronization
- 🎯 **Priority Levels**: Organize tasks by importance
- 📅 **Due Dates**: Never miss a deadline
- 🏷️ **Tags**: Categorize and filter tasks

## Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Drag & Drop**: @hello-pangea/dnd
- **Charts**: Recharts
- **State Management**: Zustand
- **Authentication**: JWT
- **Database**: JSON file storage
